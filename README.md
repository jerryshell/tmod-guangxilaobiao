## 介绍

广西老表会在你建好第二个房间后入住（第一个房间要给向导）

他会出售以下物品帮助你前期开荒：

* 南宁老友粉
* 柳州螺蛳粉
* 桂林米粉
* 窃·格瓦拉的银手镯

如果你不想花钱买食物，也可以和他战斗进行抢夺（最好确保有银套装或更好的装备再挑战）
